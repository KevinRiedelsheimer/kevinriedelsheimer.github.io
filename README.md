# Kevin Riedelsheimer's Website

Personal blog and website built with [Eleventy v3](https://www.11ty.dev/).

*Praise the Omnissiah — may your code compile and your builds succeed.* ⚙️

## Development

```bash
# Install dependencies
npm install

# Start dev server (port 5005)
npm start

# Build for production
npm run build
```

## Structure

- `content/` - Blog posts and pages (Markdown)
- `_includes/` - Nunjucks templates
- `_data/` - Site metadata and configuration
- `css/` - Stylesheets
- `public/` - Static assets
